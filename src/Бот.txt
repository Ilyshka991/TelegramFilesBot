Загружаем файлы в гугл драйв, бот автоматом будет подтягивать
Папки будут конвертироваться в кнопки, по клику на них будет переходить в подпапку

Можно написать текст:
1. Создаем index$d.txt
2. Внутри пишем текст и заливаем внутрь папки, в которой хотим показать сообщение
3. https://core.telegram.org/bots/api#markdownv2-style - текст парсится в Markdown разметке(В тг слегка урезанная версия)
4. !!!!!! In all other places characters _ * [ ] ( ) ~ ` > # + - = | { } . ! must be escaped with the preceding character \
5. Чтобы разнести текст на несколько страниц нужно создать index0.txt index1.txt. Нумерация с 0!
6. Файлы будут отображаться под последней страницей текста. Чтоб вынести на следующую нужно создать еще один index.txt файл с пустым символом

Файлы отображаются в виде ссылок
